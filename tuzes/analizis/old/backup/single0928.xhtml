Példák zárt (teljes) ortonormált rendszerekre

Észrevétel: ha y 1 , y 2 ,..., y k ,... lineárisan független olyan rendszer, hogy ℒ ( y 1 , y 2 ,... ) ¯ = X (X Hilbert-tér, a lineárisan független rendszer zárt), akkor ebből a Schmidt ortogonalizálási eljárással zárt (teljes) ortonormált rendszert kapunk.

  1. Konkrét pl: X : = L 2 ( a , b ) , ahol ( a , b ) véges intervallum.

Tétel: ebben az t ↦ 1, t ↦ t , t ↦ t 2 ,..., t ↦ t k ,... lineárisan független függvények zárt rendszert alkotnak.

Bizonyítás (vázlat): egyrészt a függvényrendszer lineárisan független: ∑ j = 0 k a j t j = 0 , . (Egy valós elsőfokú (k-ad fokú) polinomnak legfeljebb k db gyöke lehet.) Az, hogy a rendszer zárt, következik a Weierstrass approximációs tételéből. Eszerint tetszőleges f : [ a , b ] → ℝ folytonos függvényhez ∃ P k polinom sorozat, amely egyenletesen tart f-hez. Legyen g : ( a , b ) → ℝ , g ∈ L 2 ( a , b ) . A Lebesgue integrál felépítéséből kiolvasható, hogy g : [ a , b ] → ℝ folytonos függvények sűrűn vannak L 2 ( a , b ) -n. A g folytonos függvényt Weierstrass approximációs tétele szerint tetszőleges előírt pontossággal meg lehet közelíteni polinomokkal, a szuprémum normában ⇒ ezek közelítik g-t L 2 normában is.

Tétel: a fenti függvények egy zárt ortogonális rendszert alkotnak. Legyen az X alaphalmaz a valós L 2 ( 0,2 π ) . e i k t = cos ( k t ) + i sin ( k t ) , cos ( k t ) = e i k t + e − i k t 2 , sin ( k t ) = e i k t − e − i k t 2 i . Lineáris és korlátos operátorok

Állítás: legyen X, Y normált terek! Egy A : X → Y lineráis operátor folytonos ⇔ A korlátos.

Definíció: egy A : X → Y lineáris operátort korlátosnak nevezzük, ha ∃ c ≥ 0 : ‖ A x ‖ Y ≤ c ‖ x ‖ X   ∀ x ∈ X .

Tétel: legyen X normált tér, Y teljes normált tér (Banach tér), A : M → Y korlátos lineáris operátor, ahol M ⊂ X lineáris altér, de nem kell zártnak lennie. Ekkor az A-nak egyértelműen létezik korlátos lineáris kiterjesztése az M ¯ -ra (M lezárására). Más szóval: ∃ ! A ˜ : M ¯ → Y korlátos lineáris operátor, amelyre A ˜ x = A x   x ∈ M . Spec eset, mikor M ¯ = X .

Bizonyítás (vázlatos): legyen x ∈ M ¯ . Ehhez ∃ x k ∈ M : lim ( x k ) = x . Tekintsük az ( A x k ) k ∈ ℕ sorozatot Y-ban! Belátjuk, hogy ez Cauchy sorozat. ‖ A x k − A x l ‖ Y = ‖ A ( x k − x l ) ‖ Y ≤ c ⋅ ‖ x k − x l ‖ X . Legyen ε > 0 , ∃ k 0 : ∀ k , l > k 0 esetén ‖ x k − x l ‖ < ε ⇒ ‖ A x k − A x l ‖ ≤ c ⋅ ε .Y teljes ⇒ ∃ y ∈ Y : lim ( A x k ) = y . y csak x-től függ, nem függ ( x k ) -tól és egyértelmű. A ˜ ( x ) : = y , A ˜ lineáris, korlátos (és folytonos).

Hahn-Banach tétel: legyen X Banach tér, X 0 ⊂ X valódi (zárt lineáris) altér, f : X 0 → K korlátos lineáris funkcionál (azaz számértékű operátor). Ekkor ∃ f ˜ : X → K korlátos lineáris kiterjesztés, és ‖ f ˜ ‖ = ‖ f ‖ .

Korlátos lineáris funkcionálok, duális tér (Hilbert tér esetén)

Észrevétel: legyen X Hilbert tér, y ∈ X tetszőleges rögzített elem. Értelmezzük az f : X → K , f ( x ) : = 〈 x , y 〉 funkcionált.

Állítás: ekkor f korlátos lineáris funkcionál. f linearitása triviális, és folytonos is, ugyanis | f ( x ) | = | 〈 x , y 〉 | ≤ ‖ x ‖ ⋅ ‖ y ‖ .

Tétel (Riesz): legyen X Hilbert tér (valós vagy komplex), f egy korlátos lineáris funkcionál X-en. Ekkor létezik egyetlen y ∈ X , hogy f ( x ) = 〈 x , y 〉   ∀ x ∈ X .

Bizonyítás: jelöljük X 0 : = { x ∈ X : f ( x ) = 0 } -vel f magterét. X 0 altér X-ben, azaz az algebrai műveletek nem vezetnek ki X 0 -ból, és zárt részhalmaz X-ben. Utóbbi igaz, mivel f folytonos, azaz ha x k ∈ X 0 , ( x k ) → x ⇒ x ∈ X 0 , így jelen esetben f ( x k ) = 0 , f ( x k ) → f ( x ) ⇒ f ( x ) = 0 .

  1. Ha X 0 = X , f ( x ) = 0   ∀ x ∈ X , triviális eset. Ekkor legyen y = 0 .
  2. X 0 valódi altér ⇒ (Riesz-féle felbontási tétel) ∃ x 1 ≠ 0 : x 1 ∈ X 0 ⊥ . Legyen x ∈ X tetszőleges, tekintsük az X ∍ y 1 : = f ( x ) x 1 − f ( x 1 ) x elemet. Ekkor f ( y 1 ) = f ( x ) f ( x 1 ) − f ( x 1 ) f ( x ) = 0 ⇒ y 1 ∈ X 0 ⇒ 〈 y 1 , x 1 〉 = 0 .
